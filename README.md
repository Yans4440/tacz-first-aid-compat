# English

**TACZ × First Aid Compatibility**

This addon integrates TACZ with First Aid, providing a more realistic and immersive combat experience.

### Features

* Bullet hits from TACZ now correctly affect specific body parts in First Aid.
* Players in the Near Death state can no longer use weapons.
* Instant death when the head is completely destroyed.
* Full Russian language localization support.

This addon is designed to make firefights more tactical and realistic by combining TACZ's firearm system with First Aid's advanced injury mechanics.

## Русский

**Совместимость TACZ × First Aid**

Этот аддон объединяет TACZ и First Aid, обеспечивая более реалистичный и захватывающий боевой опыт.

### Возможности

* Попадания пуль из TACZ теперь корректно учитывают часть тела, в которую было произведено попадание, через систему First Aid.
* Игроки в предсмертном состоянии больше не могут пользоваться оружием.
* При полном разрушении головы игрок мгновенно погибает.
* Поддержка полного перевода на русский язык.

Аддон делает перестрелки более тактическими и реалистичными, объединяя оружейную систему TACZ с продвинутой системой ранений First Aid.
